Introduction

Python, a high-level programming language, is celebrated for its simplicity, versatility, and user-friendliness. Designed with readability in mind by Guido van Rossum in the late 1980s, Python's straightforward syntax facilitates swift development and fosters collaboration. Its wide-ranging applications span web development, data science, artificial intelligence, and more. Python's interpretive nature allows for rapid testing and iteration, while its robust community and extensive library ecosystem offer ample support for developers at all levels. Whether you're a seasoned coder or a beginner, Python provides a welcoming and powerful platform for transforming ideas into reality.

Python is known for its:

  1. Simplicity and Readability: Python emphasizes simplicity and readability, making it easy to learn and understand. Its clear and concise syntax reduces the cost of program maintenance and development.
  2. Versatility: Python is a versatile language used for a wide range of applications, including web development, data analysis, artificial intelligence, scientific computing, and more. Its extensive standard library and rich ecosystem of third-party packages contribute to its versatility.
  3. Interpretive Nature: Python is an interpreted language, which means that code written in Python is executed line by line by an interpreter, allowing for quick development and testing cycles. This interpretive nature also makes Python platform-independent, as the same code can run on different operating systems without modification.
  4. Strong Community and Support: Python has a vibrant and active community of developers worldwide. This community contributes to the language's growth and evolution through the development of libraries, frameworks, and tools. The availability of extensive documentation, tutorials, and online resources makes it easy for beginners to get started and for experienced developers to find solutions to their problems.

Variables

Variables in Python are used to store and manage data. They act as containers that hold information, such as numbers, text, or other types of data.

In this example, we declare a variable named age and assign it the value 24. We then print the value of the age variable using the print() function.

# Define an integer variable named 'age' and assign it the value 24
age = 24

# Print the value of the 'age' variable
print(age)  # Output: 24

Variables in Python can hold different types of data, including integers, floats, strings, and more. They provide flexibility and allow us to work with data in our programs.

In this example, we declare a variable named name and assign it the value 'Charlotte', which is a string. We then print the value of the name variable.

# Define a string variable named 'name' and assign it the value 'Charlotte'
name = 'Charlotte'

# Print the value of the 'name' variable
print(name)  # Output: Charlotte
Variable Scope

Variable scope refers to the area of a program where a variable is accessible. In Python, variables can have either local scope or global scope.

Local Scope:

Variables that are defined inside a function have local scope. This means they can only be accessed within that function. Once the function finishes executing, the variables cease to exist.

In this example, x has local scope and is accessible only within the my_function() function. Trying to access x outside of the function will result in an error.

def my_function():
    x = 10
    print("Inside the function:", x)

my_function()
    # Uncommenting the line below will result in an error
    # print("Outside the function:", x)

Global Scope:

Variables that are defined outside of any function have global scope. They can be accessed from anywhere in the program, including inside functions.

In this example, y has global scope and can be accessed both inside and outside of the another_function() function.

y = 20

def another_function():
    print("Inside the function:", y)

another_function()
print("Outside the function:", y)

Using Global Variables Inside Functions:

Although global variables can be accessed inside functions, if you want to modify a global variable from within a function, you need to use the global keyword.

In this example, we use the global keyword to modify the value of the global variable z from within the modify_global() function.

z = 30

def modify_global():
    global z
    z = 40
    print("Inside the function:", z)

modify_global()
print("Outside the function:", z)

While global variables can be convenient, it's generally recommended to minimize their use, as they can make code harder to understand and maintain. Instead, prefer passing variables as arguments to functions when possible.

Data Types

In Python, data types represent the type or nature of data that variables can hold. Common data types include integers, floats, strings, booleans, lists, tuples, dictionaries, sets, and NoneType. The type() function is used to determine the data type of a variable.

  1. Integer (int): Represents whole numbers without any decimal point, such as 10, -5, or 0.
  2. Float (float): Represents numbers with decimal points, such as 3.14, -0.5, or 2.0.
  3. String (str): Represents a sequence of characters enclosed within single ('') or double ("") quotes, such as "Hello", 'Python', or "123".
  4. Boolean (bool): Represents a binary value indicating either True or False.
  5. List (list): Represents an ordered collection of items enclosed within square brackets ([]), where each item can be of any data type.
  6. Tuple (tuple): Represents an ordered and immutable collection of items enclosed within parentheses (()), similar to lists but cannot be modified after creation.
  7. Dictionary (dict): Represents a collection of key-value pairs enclosed within curly braces ({}) where each key is associated with a value.
  8. Set (set): Represents an unordered collection of unique items enclosed within curly braces ({}) where each item must be unique.
  9. NoneType (None): Represents the absence of a value or a null value. It is often used to indicate that a variable has no value assigned to it.
# Integer
x = 10
print(type(x))  # Output: <class 'int'>

# Float
y = 3.14
print(type(y))  # Output: <class 'float'>

# String
name = "Katie"
print(type(name))  # Output: <class 'str'>

# Boolean
is_valid = True
print(type(is_valid))  # Output: <class 'bool'>

# List
numbers = [1, 2, 3, 4, 5]
print(type(numbers))  # Output: <class 'list'>

# Tuple
coordinates = (10, 20)
print(type(coordinates))  # Output: <class 'tuple'>

# Dictionary
person = {"name": "Bob", "age": 30}
print(type(person))  # Output: <class 'dict'>

# Set
unique_numbers = {1, 2, 3, 4, 5}
print(type(unique_numbers))  # Output: <class 'set'>

# NoneType
value = None
print(type(value))  # Output: <class 'NoneType'>
Conditional Statements

Conditional statements are essential in programming to control the flow of execution based on certain conditions. In Python, if, else, and elif are used for this purpose.

1. 'if' Statement

The 'if' statement allows you to execute a block of code if a certain condition is true.

In this example, the code inside the 'if' block will only execute if the condition x > 5 evaluates to 'True'.

x = 10

if x > 5:
    print("x is greater than 5")

2. 'else' Statement

The 'else' statement is used in conjunction with 'if' to execute a block of code when the if condition is false.

If the condition in the 'if' statement is not met, the code inside the 'else' block will be executed.

x = 3

if x > 5:
    print("x is greater than 5")
else:
    print("x is not greater than 5")

3. 'elif' Statement

The 'elif' statement stands for "else if" and is used to check for additional conditions after the initial 'if' condition is false.

In this example, if 'x' is not greater than 10, it checks if 'x' is greater than 5. If this condition is true, the corresponding block of code is executed. If none of the conditions are met, the 'else' block is executed.

x = 7

if x > 10:
    print("x is greater than 10")
elif x > 5:
    print("x is greater than 5 but not greater than 10")
else:
    print("x is not greater than 5")
Loops

1. 'for' Loop

The 'for' loop in Python is used to iterate over a sequence (such as a list, tuple, dictionary, or string) or any iterable object. It executes a block of code repeatedly for each item in the sequence.

for item in sequence:
    # code block to be executed for each item

Example:

# Iterating over a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
This loop will print each fruit in the 'fruits' list.

2. 'while' Loop

The 'while' loop in Python is used to repeatedly execute a block of code as long as a condition is true. It continues looping as long as the specified condition evaluates to 'True'.

while condition:
    # code block to be executed as long as the condition is true

Example:

# Using a while loop to count from 1 to 5
count = 1
while count <= 5:
    print(count)
    count += 1

'for' and 'while' loops are fundamental constructs in Python for iterating over sequences and executing code repeatedly based on certain conditions. They provide powerful tools for automating repetitive tasks and controlling program flow.

Functions

Introduction

Functions are blocks of organized, reusable code that perform a specific task. They provide modularity and abstraction, making your code more readable, maintainable, and efficient.

Syntax

def function_name(parameters):
    """docstring"""
    # Function body
    return [expression]
  • 'def': Keyword indicating the start oof a function definition.
  • 'function_name': Name of the function, should be descriptive and follow naming conventions.
  • 'parameters': Input values passed to the function.
  • 'doctstring': Optional documentation string that describes the function's purpose, parameters, and return value(s).
  • 'return': Keyword followed by an expression, specifies the value(s) returned by the function.
def greet(name):
    """Greet a person by name."""
    return f"Hello, {name}!"

print(greet("Charlotte"))  # Output: Hello, Charlotte!

Parameters

Functions can accept zero or more parameters.

def add(a, b):
    """Add two numbers."""
    return a + b

print(add(3, 5))  # Output: 8

Default Parameters

You can assign default values to parameters.

def power(base, exponent=2):
    """Raise a number to the power of exponent."""
    return base ** exponent

print(power(3))       # Output: 9 (3^2)
print(power(2, 3))    # Output: 8 (2^3)

Keyword Arguments

You can pass arguments by explicitly naming the parameter.

def divide(dividend, divisor):
    """Divide the dividend by the divisor."""
    return dividend / divisor

print(divide(dividend=10, divisor=2))  # Output: 5.0

Variable-length Arguments

Functions can accept a variable number of arguments using '*args' and '**kwargs'.

def concatenate(*args):
    """Concatenate all arguments into a single string."""
    return ''.join(args)

print(concatenate("Hello", ", ", "world!"))  # Output: Hello, world!

Lambda Functions

Lambda functions are anonymous functions defined using the 'lambda' keyword.

multiply = lambda x, y: x * y

print(multiply(3, 4))  # Output: 12
